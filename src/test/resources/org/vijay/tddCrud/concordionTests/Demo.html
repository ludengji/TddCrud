This is an example of spring + hibernate application. This is a demo of how to write acceptance tests for this application.

An Example

Add a customer with values: number = 50, name = comp50, address = myStreet50, mobile = 50.
Now:
  1. View the information and verify that it is same .
  2. Change the name = comp51, address = myStreet51, mobile = 51. View the information to verify the change is done .
  3. Delete this Customer and ensure View does not findit.
